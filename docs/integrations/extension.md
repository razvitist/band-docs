Ты можешь установить расширение, которое контролирует какие сайты ты посещаешь и какие видео смотрите. Расширение для браузера отслеживает, совершает ли человек полезные действия, и отправляет эти данные на браслет. Если ты хочешь изучить Python, то просто вбей слово в расширение и браслет будет поощерять просмотр видео по Python и посещение сайтов по программированию.

Установить расширение для Chrome (скоро).
